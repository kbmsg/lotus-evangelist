Cloudflare Worker Proxy

When running on GitHub Pages, all API calls go through your Cloudflare Worker to avoid CORS errors. Deploy worker.js to Cloudflare and paste the URL below (e.g. https://lotus-proxy.yourname.workers.dev). Leave blank to call APIs directly (works in Claude.ai artifacts only).

Worker routes expected:

POST {worker}/anthropic -- Anthropic API proxy

POST {worker}/buffer -- Buffer write (create/queue updates)

GET {worker}/buffer-get -- Buffer read (profiles list)

Anthropic API Key

Get your key at console.anthropic.com. Stored in your browser only.
